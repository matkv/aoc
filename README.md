## Daily Template

- Create subfolder called "day-1" for example
- Create new file "firsttask.go"
- Add this:

```go
package main

import "fmt"

func main() {
	fmt.Print("Hello world!")
}
```

- To run the file have it selected in VSCode and just press the "Run" button
- Repeat every day
